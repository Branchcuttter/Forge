FORGE free

FORGE is a free, open-source multi-agent app builder. You describe what you want to build in plain English, and a team of AI agents — each powered by free APIs — plans, designs, codes, and reviews your app. You download it as a single HTML file.

No subscription. No credit card. No install. Just free API keys and a browser.

open source

FORGE is a single HTML file. No backend, no server, no data collection. Your API keys are stored only in your browser's localStorage and are never sent anywhere except directly to the provider's API.
